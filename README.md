# Usenet Newsreader

A minimal text-only Usenet newsreader with a browser-based interface.

## Features

- Browse text-only Usenet newsgroups
- Read articles in a clean, minimal interface
- Connect to any NNTP server
- Terminal-inspired dark theme

## Installation

```bash
npm install
```

## Usage

Start the server:

```bash
npm start
```

Then open your browser to `http://localhost:3000`

## Configuration

By default, the newsreader connects to `news.eternal-september.org` on port 119. You can change the server in the UI.

## Requirements

- Node.js 14+
- Access to an NNTP server (many public servers are available)
